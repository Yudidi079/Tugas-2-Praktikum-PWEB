## Penjelasan Source Code
Pada proyek kali ini, saya membuat sebuah LinkTree menggunakan HTML dan CSS tentang Endministrator dari game Arknights Endfield. Seperti biasa, untuk awalan kode nya yaitu menggunakan kerangka HTML yaitu DOCTYPE html. Selanjutnya adalah membuat bagian kepala dari HTMLtersebut dengan menggunakan kode head dan meggunakan title sesuai dengan tema yaitu Endmin Linktree.

Pada bagian CSS yang pertama, saya menggunakan margin, padding, box-sizining, dan font-family. Gunanya adalah untuk merapikan juga untuk mengubah font default menjadi font yang kita inginkan.
Masuk kebagian body, disini saya menggunakan div class untuk mengelompokkan class. Class yang pertama adalah class container untuk bagian poto profil. Di class container, saya menggunakan head yang didalamnya saya menggunakan kode img src, h1, dan p. Untuk body dibagian CSS nya saya menggunakan berbagai macam kode,salah satu nya yaitu background-image. Pada background image ini saya meletakkan gambar background kotak-kotak berwarna putih dan silver. Agar background kelihatan rapi dan tidak kemana-mana, saya menggunakan kode seperti, background-size, background-position, background-repeat, dll, kalian bisa lihat sendiri kode nya seperti apa di HTML nya. Untuk kode dibagian .container {, saya hanya menggunakan width: 100%, max-width: 400px dan text-align: center.

Pada bagian .profile-img {, saya menggunakan width: 96px, height: 96px, border-radius: 50%, margin-bottom: 15px. border: 3px solid #fff, box-shadow: 0 4px 10px rgba(0, 0, 0, 0.1). Untuk bagian h1 {, saya menggunakan font-size: 1.5rem, color: #333, margin-bottom: 5px. Untuk bagian p {, saya menggunakan font-size: 0.9rem, color: #666, margin-bottom: 30px. 

Selanjutnya kebagian main class, yaitu class link, disini saya menambahkan 4 link yaitu link dari Instagram, Tiktok, Youtube, dan WhatsApp. Untuk bagian CSS nya, agar terlihat menarik saat menekan icon nya, saya menambahkan beberapa kode seperti kode untuk mengubah tampilan icon saat ditekan atau saat cursour berada di atas icon nya, saya menambahkan warna hitam dan untuk icon nya, saya menambahkan warna putih. Bagian-bagian terakhir nya tinggal dirapikan sedikit agar terlihat enak dilihat.
